CorePlugin

Y

加载器

Y.add

添加需要加载的js文件

调用:

Y.add(name,options);

参数:

  1. name-(string) 模块名
  2. options-(object) js文件相关数据。
    • path-(string) 文件地址
    • charset-(string) 文件编码。
    • callback-(function) 回调函数
    • requires-(array) 依赖的模块名或者文件地址

例子:

Y.add('yokaPublic',{path:'http://p1.yokacdn.com/pic/div/2012/products/public/js/yokaPublic.js'});
Y.add('fashion',{path:'http://p1.yokacdn.com/pic/div/2012/products/fashion/js/fashion.js',requires:['yokaPublic']});
Y.add('prettify',{path:'js/prettify.js'});

// console.log('添加了yokaPublic.js')
// console.log('添加了fashion.js，并且依赖yokaPublic.js')
// console.log('添加了prettify.js')

Y

调用已加载完成js文件中的方法

调用:

Y(name,function(){
	// you code
});

Y(name1,name2,...,nameN,function(){
	// 依赖多模块才能执行的方法，调用方式。
})

参数:

  1. name-(string) 模块名。多模块名已,隔开
  2. function-(function) 需要执行的方法

例子:

Y('yokaPublic',function(){
	console.log('yokaPublic is loaded!');
	yo.focus('focusId');
	
	// console.log('yo.focus 是yokaPublic.js 中的方法');
});

Y.add('prettify','fashion',function(){
	prettyPrint();
	
	// console.log('多模块加载完成后，才能调用的方法示例！')
});
