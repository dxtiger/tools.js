Domready

dom,ready时调用传入此方法中所有方法. 在domready之后调用此方法,传入的方法立即执行

调用:

Domready(function);

参数:

  1. function-(function) 方法名或方法

例子:

Domready(function(){
	console.log('noname is run')
});

function b(){
	console.log('b is run')
}
Domready(b);
