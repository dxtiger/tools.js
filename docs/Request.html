CorePlugin

Request

jsonp,ajax请求方法

Request

创建一个新请求

调用:

Request();

返回:

  • object-(object) 请求实例

例子:

javascript
var a = Request();
console.log(a); // object

Request.jsonp

执行jsonp请求

调用:

Request.jsonp(options);

参数:

  1. options-(object) json格式的样式及样式值. {name:value}
    • url-(string) 请求地址
    • async-(boolean) true/false 是否异步.默认为true
    • success-(function) 回调方法
    • data-(object) 发送的数据,json格式.如:{name:tiger}

例子:

javascript
var a = Request();

a.jsonp({
	url: 'http://www.google.com',
	success: function(){
		console.log('jsonp is complete!')
	},
	data:{name:'tiger'}
})

// jsonp is complete!

Request.ajax

执行ajax请求

调用:

Request.ajax(options);

参数:

  1. options-(object) json格式的样式及样式值. {name:value}
    • url-(string) 请求地址
    • async-(boolean) true/false 是否异步.默认为true
    • avatar-(string/object/number) 请求标记,用于判断执行的回调是否是当前请求.
    • success-(function) 成功回调方法
      • responseData 返回的数据
      • dataType-(string) 数据类型.text/xml
      • avatar 如果存在,则当作第三个参数返回
    • error-(function) 失败回调方法
      • status-(string) 失败信息
    • data-(object) 发送的数据,json格式.如:{name:tiger}
    • type-(string) 请求类型,get/post.默认get
    • dataType-(string) 返回数据类型,text/xml.默认text

例子:

javascript
var a = Request();

a.ajax({
	url: 'http://www.google.com',
	type : 'get',
	success: function(){
		console.log('jsonp is complete!')
	},
	data:{name:'tiger'},
	error : function(info){
		console.log(info);
	}
})

// ajax is complete!
