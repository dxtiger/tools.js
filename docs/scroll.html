CorePlugin

scroll

滚动

scroll

实例化

调用:

yo.scroll(options);

参数:

  1. id-(string) dom节点id
  2. boxTag-(string) 移动的容器节点名称；
  3. singleTag-(string) 内容的容器节点名称
  4. leftBtn-(string) 向前翻的按钮id
  5. rightBtn-(string) 向后翻的按钮id
  6. path-(string) 滚动方向
  7. setp-(number) 移动的步数(1个singleTag为1步)，默认为1；
  8. loop-(boolean) 是否循环，默认false
  9. auto-(boolean) 是否自动播放，默认false
  10. time-(number) 自动播放间隔时间，默认5000
  11. callback-(function) 回调方法，传入当前屏数。

返回:

  • setChange-(function) 控制滚动图变化到某一屏
    • n-(number) 第n屏
  • range-(array) 可滚动的最小值，最大值。[min,max]
  • current-(number) 当前屏数

例子:

javascript
var a = yo.scroll({
		id : 'hotNews',
		boxTag : 'div',
		singleTag : 'ul',
		path : 'top',
		step : 1,
		auto : true,
		time : 3000
});
console.log(a); // object;

scroll.setChange

控制滚动图变化到某一屏

调用:

scroll.setChange(n);

参数:

  1. n-(number) 第n屏;

例子:

javascript
var a = yo.scroll({
		id : 'hotNews',
		boxTag : 'div',
		singleTag : 'ul',
		path : 'top',
		step : 1,
		auto : true,
		time : 3000
});

a.setChange(3);

scroll.range

获取可滚动的边界值,[min,max]

调用:

scroll.range;

返回:

  • [min,max]-(array) 滚动的边界值数组
    • min-(number) 最小值
    • max-(number)

例子:

scroll.current

当前屏数

调用:

scroll.current;

例子:

javascript
var a = yo.scroll({
		id : 'hotNews',
		boxTag : 'div',
		singleTag : 'ul',
		path : 'top',
		step : 1,
		auto : true,
		time : 3000
});

a.setChange(3);
console.log(a.current); // 3


a.setChange(1);
console.log(a.current); // 1

		 	 current,

		 	var str = '';

		 		}
		 	})
		 }
		 function cb(n){
		 	spans[current].className = '';
		 	current = n;
