CorePlugin

tagSwitch

切换,用法同老版

tagSwitch

实例化

调用:

tagSwitch(tit,box,s,fn,time,show);

参数:

  1. tit-(string) 标签id/标签节点nodeName
  2. box-(string) 容器id/容器节点nodeName
  3. s-(string) 事件类型，默认mouseover
  4. fn-(function) 回调方法
    • ts[i]-(element) 当前选择的标签节点
    • bs[i]-(element) 当前选择的内容节点
    • i-(number) 当前选择的序号
  5. time-(number) 自动切换时间
  6. show-(number) 初始的选中状态，默认0

例子:

javascript
tagSwitch('a/div','b/ul');

tagSwitch('a/div','b/ul','click',callback,5000,3);
