CorePlugin

Element

dom节点的相关操作

Element.hasClass

判断此节点是否包含样式名

调用:

Element.hasClass(obj,className);

参数:

  1. obj-(element) dom节点
  2. className-(strong) 样式名。

返回:

  • (boolean) true/false

例子:

html
<div id="div" class="name"></div>
javascript
var div = document.getElementById('div');
var a = Element.hasClass(div,'name');
var b = Element.hasClass(div,'asa');

console.log(a); // true
console.log(b); // false

Element.addClass

给节点添加样式

调用:

Element.addClass(obj,className);

参数:

  1. obj-(element) dom节点
  2. className-(strong) 样式名。

返回:

  • (null) null

例子:

html
<div id="div" class="name"></div>
javascript
var div = document.getElementById('div');

Element.addClass(div,'c');
console.log(div.className); // name c

Element.addClass(div,'asa');
console.log(div.className); // name c asa

Element.removeClass

删除节点相应样式

调用:

Element.removeClass(obj,className);

参数:

  1. obj-(element) dom节点
  2. className-(strong) 样式名。

返回:

  • (null) null

例子:

html
<div id="div" class="name c asa"></div>
javascript
var div = document.getElementById('div');

Element.removeClass(div,'c');
console.log(div.className); // name asa


Element.removeClass(div,'asa');
console.log(div.className); // name

Element.removeClass(div,'bbb');
console.log(div.className); // name

Element.getStyle

获取节点相应样式

调用:

Element.getStyle(obj,style);

参数:

  1. obj-(element) dom节点
  2. style-(strong) 样式名。

返回:

  • (string) 样式的值
  • (undefined) 未取到样式的值

例子:

html
<div id="div" style="width:100px;height:100px;background:#fff;margin:0 0 10px 0;"></div>
javascript
var div = document.getElementById('div');

var width = Element.getStyle(div,'width');
console.log(width); // 100


var height = Element.getStyle(div,'height');
console.log(height); // 100

var background = Element.getStyle(div,'background-color');
console.log(background); // rgb(255,255,255)


var margin = Element.getStyle(div,'margin');
console.log(margin); // 0px 0px 10px 0px

Element.setStyle

为节点添加样式

调用:

Element.setStyle(obj,style);

参数:

  1. obj-(element) dom节点
  2. style-(object) 样式值，json格式 {name:value}。如： {left:100}
    • name-(string) 样式名
    • value-(string) 样式值

例子:

html
<div id="div" ></div>
javascript
var div = document.getElementById('div');

Element.setStyle(div,{'width':100});
console.log(div.style.width); // 100px


Element.setStyle(div,{'height':100});
console.log(div.style.height); // 100px

Element.getStyle(div,{'background':'#fff',margin:'0 0 10px 0'});
console.log(div.style.cssText); // width:100px;height:100px;background:#fff;margin:0 0 10px 0;

Element.getPosition

获取节点的相对于页面的坐标值

调用:

Element.getPosition(obj);

参数:

  1. obj-(element) dom节点

返回:

  • {x:number,y:number}(object) 坐标值
    • x-(number) x坐标值
    • y-(number) y坐标值

例子:

html
<div id="div" style="position:absolute;left:100px;top:100px;" ></div>
javascript
var div = document.getElementById('div');

var p = Element.getPosition(div);
console.log(p); // {x:100,y:100}

Element.getChild

获取节点的子节点

调用:

Element.getChild(obj,node);

参数:

  1. obj-(element) dom节点
  2. node-(string) 节点类型

返回:

  • [element,element,...element](array) 所有子节点数组
  • [element,element,...element](array) 所有node子节点数组

例子:

html
<div id="div" >
	<a href="#"></a>
	<a href="#"></a>
	<a href="#"></a>
	<div></div>
	<div></div>
</div>
javascript
var div = document.getElementById('div');

var childs = Element.getChild(div);
var divs = Element.getChild(div,'div');
var as = Element.getChild(div,'a');


console.log(childs); // [a,a,a,div,div]
console.log(divs); // [div,div]
console.log(as); // [a,a,a]
