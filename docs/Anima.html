CorePlugin

Anima

动画方法

Anima

创建一个新动画

调用:

Anima(obj/id,options);

参数:

  1. obj-(element) dom节点
  2. id-(string) dom节点的id
  3. options-(object) json格式的相应参数
    • time-(number) 动画时长,单位ms。默认500.
    • trans-(number) 动画曲线。默认1，余弦曲线。 可选：0，1，2，3

返回:

  • object-(object) 动画实例

例子:

html
<div id="div" class="name"></div>
javascript
var a = Anima('div');
console.log(a); // object

Anima.start

开始执行动画

调用:

Anima.start(options);

参数:

  1. options-(object) json格式的样式及样式值. {name:value}
    • name-(string) 样式名，如: width, left
    • value-(string/number) 样式值，如：100px等

例子:

html
<div id="div" class="name"></div>
javascript
var a = Anima('div');

a.start({
	left: 100,
	top: 10,
	background:'#ff0'
})

Anima.cancel

取消当前动画

调用:

Anima.cancel(fn);

参数:

  1. fn-(function) 取消动画后，回调的方法

例子:

html
<div id="div" class="name"></div>
javascript
var a = Anima('div');

a.start({
	left: 100,
	top: 10,
	background:'#ff0'
})

a.cancel(function(){
	alert('anima is cancel')
})

Anima.complete

动画完成后，执行的方法

调用:

Anima.complete(fn);

参数:

  1. fn-(function) 方法名或方法。动画完成后，执行一次传入的方法。

例子:

html
<div id="div" class="name"></div>
javascript
var a = Anima('div');

a.start({
	left: 100,
	top: 10,
	background:'#ff0'
})
a.complete(function(){
	console.log('anima is complete!')
})

// anima is complete!
