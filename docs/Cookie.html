CorePlugin

Cookie

cookie信息的相关方法

Cookie.read

读取cookie的值

调用:

Cookie.read(name);

参数:

  1. name-(string) cookie名

返回:

  • value-(string) cookie值
  • null-(null) 没有值的时候返回null

例子:

javascript
var a = Cookie.read('aaa');
console.log(a); // null;

Cookie.write

写入cookie

调用:

Cookie.write(options);

参数:

  1. options-(object) 写入的cookie信息,json格式;
    • name-(string) 名字
    • value-(string) 值
    • domain-(string) 所在域
    • path-(string) 所在目录
    • day-(number) 保存时间.单位:天

例子:

javascript
Cookie.write({
	name : 'aaa',
	value : '111',
	day : 1
});

var a = Cookie.read('aaa');
console.log(a); // 111;

Cookie.dispose

删除cookie

调用:

Cookie.dispose(name);

参数:

  1. name-(string) cookie名

例子:

javascript
Cookie.write({
	name : 'aaa',
	value : '111',
	day : 1
});

var a = Cookie.read('aaa');
console.log(a); // 111;



Cookie.dispose('aaa');
console.log(a); // null
